This is a list of changes on this website. If there's any significant update to the contents, they will be probably shown here. Additionally, you can monitor commit changes on GitHub to see all exact changes. This page serves it's purpose as a summary for most important commits.

January 2025

January 30th, 2025:

  • Updated all shader files to include Depth mode. This was an oversight on my end, this must be featured by default because shadows won't cast without depth prepass. Sorry!
  • Updated Your First Unlit Shader. s&box will change the name of the main rendering mode from VrForward to Forward. Added a reference to "Relics of the Past" article. Fixed typo. Added a very little notation about forward+ rendering.
  • Shader files have also been updated to use Forward mode from now on.
  • Relics of the Past article has been updated to feature old mode VrForward and what should be used instead of it from now on.
  • Added a dedicated changelog page that you're currently reading. :-)

January 22nd, 2025:

  • Added new article: Triplanar Mapping

January 21st, 2025:

  • Added new article: Part 11. Frame Buffer
  • Added new article: Procedural Noises and Shapes
  • Added a link to a discussion thread in s&box discord on welcome page.

January 19th, 2025:

  • Implemented nav-item custom HTML tag that replaces old a tag on index page. This should make easier to maintain, edit and implement new article links in the index navigation column.
  • Added notranslate to all material icons so they wouldn't be affected by translators and break how icons are rendered on page. Tested with Google Translate but I am not sure if this will work with other machine translators. Fixes issue #1 on GitHub.
  • Updated Contributing page to document new added custom tag.
  • Added a button that toggles between light and night themes
  • Bunch of CSS stylesheet re-organization to improve amount of content that can be adjusted for custom themes
  • Some little JS code cleanup, new comments and removal of some dead code from July 2024.

January 18th, 2025: (initial public release)

  • Added new article: Relics of the Past
  • Released a new three-part course on post-processing shaders:
    • Basic Postprocessing
    • Screenspace Fog
    • Dithering Effect
  • Implemented article-ref tag which allows linking and redirecting users to mentioned article by clicking on the tag. Works a bit smoother than a links.
  • URL now correctly updates the current state and query when switching between articles using article-ref tag on pages.
  • Scroll position for each article is now locally saved and restored every time you open this article again. This will persist between every session as it's saved in your browser's local storage.
  • Huge number of phrasing improvements and proofreading for most of the articles on website
  • Uploaded bunch of sample models and materials to use for some articles that rely on specific visuals/mesh data

January 17th, 2025:

  • Implemented new custom tag, article-header and updated the documentation to show how to correctly use this tag. Replaces previous custom div layout which was annoying to maintain.
  • Updated test page with latest custom tags
  • Implemented article navigation on the right: list of headings and subheadings are populated at runtime, and JS will automatically highlight the section you're currently reading on page. Clicking on these headings will automatically scroll your page to the vertical position of given section.

January 16th, 2025: (internal pre-release)

  • Added new article Contributing which documents most of the important stuff you need to know if you plan to edit or submit new articles on the website
  • JS: fixed innerHTML being updated twice, which was messing with custom elements and causing them to render twice on page
  • Re-enabled scrolling in the navigation bar since amount of articles won't fit on 1080p screen anymore
  • Added download-file tag
  • Current page is now correctly updated and applies the name of currently open article
  • Finished vertex displacement article and improved the wording in many other articles

December-January 15th, 2025:

  • Initial upload to GitHub
  • Finished writing first ten main articles and a few reference pages. Lots of work on styling and JS frontend script, and lots of other changes that I can't remember in details anymore..
